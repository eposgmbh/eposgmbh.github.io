Property Value

Func<TOptions, CommandLineDefinition, Task<int>>

Remarks

The command line function is called with a TOptions instance that is initialized with the command line arguments. The function is only called, if the command line can be parsed successfully. The function should return an error code (0 if successful). If the async command line function is not set, the sync version (CommandLineFunc) is called.

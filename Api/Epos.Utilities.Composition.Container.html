Remarks

Type T may also be a concrete type. If an abstraction is given, you can register a concrete implementation via the fluent interface ImplementedBy<TImplementation>(). Other options can also be specified via the fluent interface.

Examples
     Container theContainer = new Container();

     theContainer
         .Register<ITestService>()
         .ImplementedBy<TestService>()
         .WithParameter("connectionString", "Hello World!") // Constructor params
         .AndParameter("maxCount", 10)
         .WithLifetime(Lifetime.Singleton);

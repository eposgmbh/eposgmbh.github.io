Class ComponentCreationStrategy

Provides the base class for a component creation strategy.

Inheritance
Object
ComponentCreationStrategy
Namespace: Epos.Utilities.Composition
Syntax
public abstract class ComponentCreationStrategy : object
Remarks

Components can be created either by a factory method (see WithFactoryMethod<TImplementation>(Func<TImplementation>)) or by invoking a constructor with parameters (see WithParameter(String, Object)). Therefore the methods and GetInstance(ConstructorInfo, Object[]) must be implemented.
